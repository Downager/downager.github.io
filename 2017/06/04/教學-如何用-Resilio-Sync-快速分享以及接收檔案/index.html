[教學] 如何用 Resilio Sync 快速分享以及接收檔案

發表於 2017-06-04 更新於 2020-11-19 分類於 Windows 閱讀次數： Disqus： 文章字數： 210 所需閱讀時間 ≈ 1 分鐘

前言

　　原本分享檔案都是利用 HFS，但是對檔名有特殊符號的支援度不佳，以及分享大型檔案時續傳不易，決定使用 Resilio Sync 作為分享大型檔案的解決方案

操作

接收檔案

  • 首先先到官網去下載軟體
  • 自行決定是否要開機自動啟動
  • 設定你的名稱
  • 選擇 輸入金鑰或連結
  • 輸入金鑰
  • 選擇資料夾存放位置
  • 開始同步

分享檔案

  • 選擇 標準資料夾 以及選擇要分享的資料夾位置
  • 複製金鑰，有分 唯讀 或 可讀寫 兩種，單純分享檔案選擇 唯讀 即可。
    若要雙邊協作同步檔案，那就選擇 可讀寫 金鑰
