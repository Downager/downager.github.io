版本挑選

版本號說明請參考友站連結：

大軒軒的筆記本 - Cisco版本號說明12.4和15

官方工具：

  • Cisco Software Download
    輸入 Product Name 來搜尋 & 下載 IOS，沒有特別需求一般推薦選擇 Suggested Release
  • Cisco IOS Software Checker
    檢查對應版本是否有已知漏洞
  • Cisco Bug Search Tool
    輸入 Product Name & Releases，查詢是否有已知的 Bug
