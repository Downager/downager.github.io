解決方法

  1. 升級 IOS 版本
  2. 手動關閉 Smart Install
    switch(config)# no vstack
  3. 建立 Firewall Policy （禁止連線到 switch的 TCP 4786）
  4. 若必須要使用請在 Switch 上設定 ACL 限定來源
